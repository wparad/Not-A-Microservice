This is template 4. I'll define the HTML template here, filled with runtime replace fields denote by { { variablename } }. You can see this in the source, but basically it looks like this:
$scope.data = {id: 1, name: 'This is my name', text: 'hello'};

This is the id file: {{data.id}}

This is the name file: {{data.name}}

This is the text file: {{data.text}}

Maybe you are now asking yourself if you could just pull this information from anywhere? The answer is yes with template 5.
